# Cookie Clicker Autoclicker
---
This is a script that can autoclick for you in cookie clicker. To run it open the console in dev tools and paste it in. You may have to type allow pasting to allow pasting.

Note: This will not give you the "cheated cookies taste awful" achievement. It also might also buy the occult obstruction debug upgrade which will stop cookie production, however you can unlearn occult obstruction.
